Небольшая программа, которая находит максимальное значение в слайсе целых положительных чисел.

Функции:
generateRandomElements() - Функция генерирует числа и записывет в слайс. Функция принимает размер слайса и возвращает слайс с данными.
maximum() - Эта функция находит максимальное число в слайсе, который передаётся в функцию, и возвращать его. Для этой функции имеется unit-тест в файле main_test.go
maxChunks() - Функция нахождения максимумов в частях слайса.
